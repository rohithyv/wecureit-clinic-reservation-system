# WeCureIT: Clinic Reservation & Management System

**WeCureIT** is a full-stack healthcare management platform designed to streamline interactions between patients, doctors, and medical facilities. The system handles complex scheduling logic, multi-facility authorization and secure clinical record-keeping.

---

## 🛠️ Technology Stack

* **Frontend:** React.JS
* **Backend:** Java 17, Spring Boot 3.x
* **Database:** PostgreSQL 14+ (Schema: `wecureit`)
* **Tunnelling:** **Ngrok** (For external API access and mobile testing)

---

## 🚀 Core Features

* **Role-Based Access:** Distinct workflows and permissions for **Patients** and **Doctors**.
* **14-Day Cycle Scheduling:** Automated availability management based on a recurring bi-weekly cycle.
* **Facility & Room Management:** Integrated tracking of clinic locations and specific room assignments.
* **Clinical Notes:** Secure creation and storage of appointment metadata and doctor notes.
* **Financial Tracking:** Logic for handling base amounts, penalty fees, and overage charges.
